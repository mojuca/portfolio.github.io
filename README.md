# My Portfolio

This is my personal portfolio website, hosted on GitHub Pages.

## How to Use

1. Replace the placeholder content in `index.html` with your actual information:
   - Update the title, name, profession, location, skills, and hobbies in the About section
   - Add your real projects in the Projects section
   - Customize the contact form or replace it with your preferred contact method

2. Replace `images/profile.jpg` with your actual profile picture (keep the same filename)

3. Customize the styles in `css/style.css` to match your preferences

4. The site is ready to be hosted on GitHub Pages. Since this repository is named `portfolio.github.io`, it will be available at `https://[your-username].github.io/`

## Features

- Responsive design that works on desktop and mobile
- Smooth scrolling navigation
- Contact form (currently shows a demo alert - integrate with a backend service for real functionality)
- Clean, modern layout

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Deployment

This site is automatically deployed via GitHub Pages when you push changes to the `main` branch.
